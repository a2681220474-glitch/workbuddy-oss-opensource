# WorkBuddy OSS v0.19.0 Knowledge And RAG Mainline Start

`v0.19.0` marks the end of the `v0.18.x` deployment/runtime mainline and opens
the next major version: knowledge base and RAG formalization.

## Current Baseline

- Feishu has real receive and approval/send closure samples.
- WeCom has real encrypted callback receive and real send closure samples.
- Team workbench, permissions, assignees, audit logs, and object timelines are
  available.
- Runtime stack now includes:
  - health checks
  - Alembic baseline
  - backup and restore plan
  - runtime-jobs worker
  - scheduled daily reports
  - structured logs
  - operations runbook

## v0.19.x Focus

- Knowledge detail page.
- Source references from tickets, community questions, messages, and approvals.
- Knowledge version records.
- Edit, review, publish, and archive flow.
- Knowledge hit records.
- Search and retrieval hardening.
- Later: vector retrieval / RAG.

## Verification

- `npm run build:web`

- `npm run check:wecom-acceptance`
